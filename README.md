# Titles

This is an API for getting rss titles from Google RSS

## Usage

* git clone https://github.com/husniddin96/node-rss
* npm install
* node app

## GetTitles

```bash
    curl -X GET \
    http://localhost:8080/feeds \
    -H 'Cache-Control: no-cache' \
    -H 'Connection: keep-alive' \
    -H 'Host: localhost:8080' \
    -H 'cache-control: no-cache'
```
